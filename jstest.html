Welcome to Anti36Manager

Thank you for trying out my project! If you're running this for the first time, you might encounter an error—don't worry, I'll guide you through it. If you skipped the README.md, here is a quick summary.

What is Anti36Manager?

Anti36Manager is a turbo lightweight browser-UI(ed?) tool for organizing images/videos (portrayals) in a structured way. Think of it as a gallery with more control.

Setup

Use a separate disk (e.g., a USB drive).

Create a folder named Anti36Local.

Create origin and persona folders in which you want your portrayals to land. See below

Anti36Local(1)→(n)Origins(1)→(n)Personas(1)→(n)Portrayals (You wouldn't manually place portrayals though)

A single file path might look like:

disk:/Anti36Local/Origin/Persona/index_tags_.extension

Whether you're archiving personal memories or twisted unholy portrayals, it is all up to you.

In case of an exception, You don't have to worry about losing your data. Files are moved and folders untouched.

Enjoy!
